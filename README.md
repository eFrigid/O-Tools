O-Tool
=======

Features
--------
- User friendly gui (windows and linux amd64)
- Backup and restore (adb)
- Push and Pull files
- Import camera photos
- Install apk
- Record Screen
- Shell
- Root
- Unlock Bootloader
- Lock Bootloader
- Backup EFS
- Flash zip files
- Install roms (optional backup and wipes)
- Flash boot.img files
- Install recovery
- Root mode

Devices
-----------
- OnePlus
  - One A0001 _bacon_
- Oppo
  - Find 7(a)(s) _find7 find7a_
  - Find 5 _find5_
  - N1 _n1_
