O-tools

Toolkit for Oppo and OnePlus Devices

Download this project as a .zip file Download this project as a tar.gz file

O-Tool

Features

  • User friendly gui (windows and linux amd64)
  • Backup and restore (adb)
  • Push and Pull files
  • Import camera photos
  • Install apk
  • Record Screen
  • Shell
  • Root
  • Unlock Bootloader
  • Lock Bootloader
  • Backup EFS
  • Flash zip files
  • Install roms (optional backup and wipes)
  • Flash boot.img files
  • Install recovery
  • Root mode

Devices

  • OnePlus
    • One A0001 bacon
  • Oppo
    • Find 7(a)(s) find7 find7a
    • Find 5 find5
    • N1 n1
